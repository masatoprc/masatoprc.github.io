    Technologies Used: C, Objective-C, PebbleKit SDK, BrainTree SDK

    Technologies Used: C, Objective-C, PebbleKit SDK, Facebook SDK
    Summary: • Emergency service that sends out virtual cry for help on Facebook upon shaking Pebble Watch
    • Created the iOS Companion app and Pebble Watch app

  • Event Manager (Swift iOS)

    Technologies Used: Swift, Objective-C, CoreData

    • Implemented passcode configuration, and enabled users to edit photos and share them on social media (Facebook and Twitter)
